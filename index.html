The Interactive Todo List

You can add a task by typing into an input field and clicking a button.
The new task is immediately displayed in the task list.
Tasks will be validated before being added (e.g., non-empty and unique).
You can delete tasks from the list, and the task should be removed from the DOM.
Moreover, you can click an "Edit" button for a task, modify the text, and save the changes.
